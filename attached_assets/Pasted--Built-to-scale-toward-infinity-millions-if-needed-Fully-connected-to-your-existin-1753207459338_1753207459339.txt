## 📦 Instructions (Replit Deployment)

1. Create a file inside your Replit `/kons` folder named:

   ```
   konsPowaTaskEngine.ts
   ```

2. Paste the full code above into it.

3. Import and use it anywhere in your app:

```ts
import { KonsPowaTasks, getKonsTasks } from './kons/konsPowaTaskEngine'

console.log(getKonsTasks()) // Logs all 150 divine tasks
```
