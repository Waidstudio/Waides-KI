## 📁 2. Update `WaidesKI.jsx` — Add Summon Logic

```jsx
const getAnswer = (question) => {
  const q = question.toLowerCase().trim();

  if (BotMemory.introQuestions[q]) return BotMemory.introQuestions[q];
  if (q.includes("waides ki") || q.includes("who are you")) return BotMemory.identity;
  if (q.includes("your mission") || q.includes("why were you created")) return BotMemory.tradingMission;

  // 🔥 Summon WaidBot Commands
  for (const cmd in BotMemory.summonCommands) {
    if (q.includes(cmd)) {
      return BotMemory.summonCommands[cmd];
    }
  }

  if (q.includes("wallet")) return "Open your SmaiWallet to check your ETH positions.";
  if (q.includes("price")) return "Check the Live Tracker module for real-time ETH price.";
  if (q.includes("guardian")) return "ETH Guardian is always active, watching your contracts.";

  return "I'm still learning. Ask about WaidBots, ETH, predictions, or trading.";
};
```

## 📁 3. Optional: Show WaidBot Panel When Activated

You can add visual flair with a **summoned bot UI**:

### 🆕 Inside `WaidesKI.jsx`

```jsx
const [showBot, setShowBot] = useState(false);

const handleSend = () => {
  if (!input.trim()) return;
  const response = getAnswer(input);

  // 🧠 If response includes “WaidBot activated”, show the bot panel
  if (response.toLowerCase().includes("waidbot")) {
    setShowBot(true);
  }

  const newChat = [...chat, { role: "user", text: input }, { role: "bot", text: response }];
  setChat(newChat);
  setInput("");
};
```

Then add this to the JSX:

```jsx
{showBot && (
  <div className="waidbot-panel">

    Strategy: Eternal Spiral Mode<br />

)}
```
