#### **1.2. Smaipin Redemption**

* **Frontend Logic:** The **Smaipin Redemption** section will have a text input for entering the **Smaipin** code and a button to redeem it.
* **Backend Logic:** Once the code is entered, it will make an API call to verify the **Smaipin** code, check its validity, and update the **SmaiSika** balance accordingly.

**Frontend Example:**

```jsx
const redeemSmaipin = async () => {
  const code = document.querySelector('input').value;
  const response = await fetch('/api/redeem', {
    method: 'POST',
    body: JSON.stringify({ code }),
    headers: { 'Content-Type': 'application/json' },
  });

  const data = await response.json();
  if (data.success) {
    setSmaiSikaBalance(smaiSikaBalance + data.amount);
    alert('Smaipin redeemed successfully!');
  } else {
    alert('Invalid Smaipin code.');
  }
};
```

### **2. Backend Implementation**

#### **2.1. Smaipin Redemption Endpoint**

This API endpoint will accept the **Smaipin** code, validate it, and update the **SmaiSika** balance.

```javascript
// Example with Express.js (Node.js backend)
app.post('/api/redeem', async (req, res) => {
  const { code } = req.body;
  const smaipin = await findSmaipinByCode(code); // Find Smaipin in the DB

  if (!smaipin || smaipin.redeemed) {
    return res.status(400).json({ success: false, message: 'Invalid or already redeemed code' });
  }

  // Redeem the Smaipin
  const user = await getUserById(smaipin.userId);
  user.smaiSikaBalance += smaipin.amount;
  await user.save();

  // Mark the Smaipin as redeemed
  smaipin.redeemed = true;
  await smaipin.save();

  res.json({ success: true, amount: smaipin.amount });
});
```
