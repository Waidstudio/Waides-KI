## 🧠 STEP 4: Update `WaidesKI.jsx` to Use GPT When Needed

Replace your `getAnswer()` function with:

```jsx
import { Configuration, OpenAIApi } from "openai";

// Set up OpenAI
const configuration = new Configuration({
  apiKey: process.env.OPENAI_API_KEY,
});
const openai = new OpenAIApi(configuration);

// Inside your WaidesKI component
const getAnswer = async (question) => {
  const q = question.toLowerCase().trim();

  // 🎯 FIRST: check if it's part of BotMemory
  if (BotMemory.introQuestions[q]) return BotMemory.introQuestions[q];

  // 🔮 Custom symbolic WaidBot logic
  for (const cmd in BotMemory.summonCommands) {
    if (q.includes(cmd)) return BotMemory.summonCommands[cmd];
  }

  // 🔁 SECOND: use GPT for all other questions
  try {
    const gptResponse = await openai.createChatCompletion({
      model: "gpt-3.5-turbo", // or gpt-4 if your key supports it
      messages: [{ role: "user", content: question }],
    });

    return gptResponse.data.choices[0].message.content;
  } catch (err) {
    console.error("GPT error:", err);
    return "I tried to answer using universal intelligence, but something went wrong.";
  }
};
```

## 🔁 STEP 5: Modify `handleSend()` to Support Async

Update your `handleSend` in `WaidesKI.jsx`:

```jsx
const handleSend = async () => {
  if (!input.trim()) return;

  const userText = input;
  setInput("");
  setChat((prev) => [...prev, { role: "user", text: userText }]);

  const response = await getAnswer(userText);
  setChat((prev) => [...prev, { role: "user", text: userText }, { role: "bot", text: response }]);
};
```
